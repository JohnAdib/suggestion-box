# SuggestionBox

**SuggestionBox** is a modern feedback collection platform designed to capture, manage, and analyze user feedback efficiently. Built with the latest web technologies. SuggestionBox provides a seamless experience to easily submit feedbacks as bug or suggestions.

## Technologies

### Frontend

- **Vue.js**: Reactive components and state management for a dynamic user interface.
- **Vite**: Fast and optimized build tool for modern web projects.
- **Tailwind CSS**: Utility-first CSS framework for rapid UI development.

### Backend

- **Node.js**: Scalable and efficient server-side JavaScript runtime.
- **TypeScript**: Strongly typed programming language for improved code quality and maintainability.
- **MongoDB**: NoSQL database for flexible and scalable data storage.
